# React Routing Example

This is a simple React application demonstrating routing with React Router.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173` (or the port shown in the terminal).

## Features

- Home Page
- About Page
- Navigation between pages using React Router
- Single Page Application behavior (no full page reloads)

## Concepts Covered

- React Router
- Routes and Route components
- Link component for navigation
